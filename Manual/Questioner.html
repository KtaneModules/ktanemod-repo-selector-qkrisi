On the Subject of Questioner

No, it's still not the final name.

On the module you see:
  • A display with a question on it
  • Either an A-Z keyboard or a Yes-No keyboard
If you don't, you either:
  • Clicked on the wrong manual :P
  • Have a screen covering the module saying "Please enable questionerService!". If this is the case, you most likely disabled the "questionerService" in Mod Selector. You can solve the module by clicking on the "Solve module" button.
You need to answer 3 questions, and the module will solve. If you get one wrong, the whole module resets.

Questions can be either one of the followings:
  • What is the Xth letter of the module that is Yth on the repo sorted by {sorting method}?
  • What is the Xth letter of the module that is loaded Yth in game?
  • Is {module} disabled by an enabled profile?
  • What is the Xth letter of the module that is the Yth one on a list of all disabled modules? (A-Z)
  • What is the Xth letter of the module that is the Yth one on a list of all enabled modules? (A-Z)
  • Is the Xth module on the repo sorted {sorting method} loaded in the game?
  • Is the Xth module loaded in the game (A-Z) the same module as the Xth module on the repo sorted {sorting method}?
  • What is the Xth letter of the module that is the Yth one on a list of all disabled modules sorted by {sorting method} on the repo?
  • What is the Xth letter of the module that is the Yth one on a list of all enabled modules sorted by {sorting method} on the repo?
  • What is the Xth letter of the module that is Yth on the bomb sorted {A-Z/Z-A}
Exclude vanilla modules except for the last question!

Sorting methods can be the following:
  • A-Z
  • Z-A
  • Defuser difficulty (very easy - very hard)
  • Defuser difficulty (very hard - very easy)
  • Expert difficulty (very easy - very hard)
  • Expert difficulty (very hard - very easy)
  • Publish date (newest to oldest)
  • Publish date (oldest to newest)
Each module that is loaded in the game but isn't present on the repo will have a fake profile with the following information:
Property name Value
Name ID of the module
Module ID ID of the module
Defuser Difficulty Easy
Expert Difficulty Medium
